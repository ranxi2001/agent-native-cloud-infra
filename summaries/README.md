# Workstation Summaries

This directory contains one compact, reviewed summary per ISO week under
`weekly/YYYY-Www.md`.

Update the current week's file only when work changes a durable outcome,
decision, blocker, or next action. Link task evidence and exact upstream refs
instead of copying command logs or PR timelines. Keep each summary short enough
to scan in one work loop.

Weekly summaries are the only reporting cadence owned by this workstation. Do
not create daily or monthly rollups. Raw conversations, secrets, volatile logs,
and generated output do not belong here.
