# 技术报告索引

`report/` 保存工作台自有的技术报告，独立于 `projects/` 的源代码仓库登记和 `lanes/` 的上游贡献状态。报告可以综合多个项目或公开资料；关联项目只引用报告，不拥有或复制报告内容。

## 维护边界

- 每份报告使用稳定 ID，并在正文元数据中记录状态、整理日期和证据截止时间。
- 报告中的代码或项目事实必须引用精确仓库版本；只有需要核对或修改项目源码时，才进入对应 project context。
- 项目 lane、任务证据和周报链接到本目录，不复制长篇报告。
- 报告不继承 upstream、fork、branch、worktree 或上游发布授权；需要对外发布时另走相应审批流程。
